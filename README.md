# PizzaBot

Run on Windows -->
Step 1.
Go to ...\PizzaBot\bin\Debug\net5.0

Step 2.
Run PizzaBot.exe

Run on Ubuntu -->
Step 1.
run an ubuntu docker container
docker run -it ubuntu

Step 2.
clone the project from github
apt install -y git
git clone <url> /project

Step 3.
setup execution environment
sh /project/prepare.sh

Step 4.
run the project
cd /project/PizzaBot
dotnet run
